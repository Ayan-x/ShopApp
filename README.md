
# Ecomzy

It is a frontend of E-commerce Website. This product are listed 
are from FAKE API call. 

Api Reference - https://fakestoreapi.com/products

## Installation

Install my-project with npm

```bash
  npm install
```
    
## Run Locally

```bash
  npm run start
```


## Tech Stack

React, Redux, TailwindCSS
